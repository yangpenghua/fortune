## 根因分析（BugFix）

**问题根因**：
- 原代码中按钮点击处理逻辑内联在 JSX 中，调试和维护不便
- CSS 层级未明确设置，可能在某些浏览器或场景下导致按钮被遮挡
- 缺少对按钮可点击性的明确 CSS 保障

**影响范围**：
- 仅影响前端交互，不影响后端逻辑
- 三种测算风格按钮都可能受影响（星座运势、传统八字、塔罗占卜）
